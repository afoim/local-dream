# 📱 LocalDream 提示词插件移植项目

## 🎯 项目目标

将 [sd-webui-prompt-all-in-one](https://github.com/Physton/sd-webui-prompt-all-in-one) 提示词插件移植到 LocalDream Android App,让用户可以通过点击中文标签来添加提示词,无需手动输入英文。

## ✨ 功能特性

### 核心功能
- ✅ **中英文双语显示**: 每个标签显示中文翻译和英文原文
- ✅ **三级分类结构**: 类别 > 分组 > 标签,清晰有序
- ✅ **智能搜索**: 支持中英文搜索,快速定位想要的提示词
- ✅ **一键添加**: 点击标签自动添加到输入框,自动处理逗号分隔
- ✅ **Material Design 3**: 现代化的UI设计,与App风格统一

### 提示词数据
- 📦 **数据量**: 包含数千个提示词标签
- 🏷️ **分类**: 人物、服饰、场景、风格、动作、表情等多个大类
- 🌐 **语言**: 中文界面,英文提示词
- 💾 **大小**: JSON文件约192KB

## 🔧 技术实现

### 数据流程
```
YAML文件 → Python脚本 → JSON文件 → Repository → UI
```

1. **数据源**: `zh_CN.yaml` (原始提示词数据)
2. **转换**: Python脚本将YAML转为JSON
3. **存储**: JSON文件放在 `assets/prompts/`
4. **加载**: Repository从assets读取并解析
5. **展示**: UI组件显示分类和标签
6. **选择**: 用户点击标签,添加到输入框

### 核心组件

#### 1. 数据模型 (PromptTag.kt)
```kotlin
data class PromptTag(
    val english: String,    // 英文提示词
    val chinese: String     // 中文翻译
)

data class PromptGroup(
    val name: String,       // 分组名称
    val color: String?,     // 颜色(可选)
    val tags: List<PromptTag>
)

data class PromptCategory(
    val name: String,       // 类别名称
    val groups: List<PromptGroup>
)
```

#### 2. 数据仓库 (PromptTagRepository.kt)
- 从assets加载JSON文件
- 解析JSON为数据模型
- 提供搜索功能
- 缓存数据提高性能

#### 3. UI界面 (PromptTagSelectorScreen.kt)
- 对话框形式展示
- 可折叠的分类和分组
- 搜索功能
- 标签选择和添加

2. 点击Prompt输入框旁的 `+` 按钮

5. 重复3-4步添加更多标签
6. 点击生成按钮

## 🔄 更新提示词

如果需要更新或自定义提示词:

### 方法1: 更新原始数据
1. 编辑 `demo/sd-webui-prompt-all-in-one-main/group_tags/zh_CN.yaml`
2. 运行转换脚本: `uv run convert_yaml_to_json.py`
3. 重新编译App

### 方法2: 直接编辑JSON
1. 编辑 `app/src/main/assets/prompts/zh_CN.json`
2. 重新编译App

### 方法3: 添加自定义分类
在YAML或JSON中添加新的分类:
```yaml
- name: 我的自定义分类
  groups:
    - name: 自定义分组
      tags:
        my_tag: 我的标签
        another_tag: 另一个标签
```

## 🐛 常见问题

### Q1: 编译失败?
**A**: 检查以下几点:
- 确保所有文件都已创建
- 检查导入语句是否正确
- 清理项目: Build -> Clean Project
- 重新同步Gradle

### Q2: 找不到JSON文件?
**A**: 
- 确认文件路径: `app/src/main/assets/prompts/zh_CN.json`
- 重新运行转换脚本
- 检查assets文件夹是否在正确位置

### Q3: 对话框不显示?
**A**:
- 检查状态变量是否正确声明
- 确认对话框代码在正确位置
- 查看Logcat日志

### Q4: 标签点击无反应?
**A**:
- 检查 `onTagSelected` 回调是否正确实现
- 确认 `onPromptChange` 函数正常工作
- 查看是否有异常日志

### Q5: 搜索功能不工作?
**A**:
- 检查搜索逻辑是否正确
- 确认数据已正确加载
- 尝试清除搜索框重新输入
